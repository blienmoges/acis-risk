# ACIS - Week 1: Risk Analysis & EDA

This repository contains code and analysis for Week 1 of the AlphaCare Insurance project.
Structure:
- data/                 Raw datasets (do not commit large files)
- notebooks/            Jupyter notebooks for EDA and testing
- src/                  helper functions and utilities
- figures/              EDA plots
- reports/              final slides/reports
- tests/                unit / smoke tests

Getting started:
1. Create virtual env: python -m venv .venv
2. Activate it and install deps: pip install -r requirements.txt
3. Start Jupyter: jupyter lab

KPI:
- Dev environment reproducible (requirements.txt)
- task-1 branch created, daily commits (min 3)
- At least 3 creative EDA plots produced and saved in /figures
